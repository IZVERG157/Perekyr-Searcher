**Perekyr Searcher**

**Рекурсивный искатель файлов**
-По имени
-По имени+расширению
-По расширению

**Функционал**
-Искать первый попавшийся
-Искать файлы
-Искать файлы и удалить их
-Искать файлы по расширению

**Принцип работы**
-С одной из директорий выбранной пользователем начинается поиск. 
Если файлы с введенным именем не найдены, то рекурсивно начинается поиск из этой директории в других директориях. 
Функция вызывает сама себя для дальнешего рекурсивного поиска.
-Получения файлов и директорий в директории осуществляется с помощью Directory.EnumerateFiles() & Directory.EnumerateDirectories()
для более быстрой работы.
